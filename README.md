# Form Validation Mini Project

This is a simple form validation mini project built using **HTML, CSS, and JavaScript**.  
The project focuses on validating user input on the client side and improving form usability.

##  Features
- Input validation using JavaScript
- Required field checking
- Basic error message display
- Clean and simple UI using CSS
- Beginner-friendly code structure

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
